black-chat — это мессенджер с серверной частью на базе фреймворка <a href="https://laravel.com/">Laravel</a>. Клиентская часть(web-браузер) написана на <a href="https://react.dev/">React</a>. На данном этапе реализован обмен текстовыми сообщенями через отдельный <a href="http://socketo.me">websocket</a>-сервер и видеозвонки(почти) через встроенный <a href="https://webrtc.org">WebRTC</a> браузера и <a href="https://github.com/coturn/coturn/tree/master">coturn</a>. <br>
Этот проект можно рассматривать как учебный — для отработки навыков совместной разработки. Сейчас много работы как по фронту, так и по бэку — по фронту неплохо было бы в первую очередь реализовать адекватный UI для звонков(отображение уведомления о входящем звонке, подстроить разрешение захвата видео под разрешение экрана собеседника на этапе согласования параметров звонка; кроме того есть потребность в дизайнерских решениях — иконки, анимации загрузки), по бэку надо прописать авторизацию действий пользователей, так как её нет от слова совсем😁. Также надо реализовать групповые чаты, хранение изображений и в перспективе чего-нибудь еще.<br>
